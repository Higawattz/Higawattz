## How I Am Building This Portfolio

Each serious project in this profile should answer five questions:

1. **What problem does it solve?**
2. **Who is it for?**
3. **How does it work?**
4. **How can someone run or review it?**
5. **What technical decisions were made and why?**

This makes the profile easier to validate for recruiters, hiring managers, and engineers reviewing asynchronously.
