# Portfolio Projects

This folder contains non-confidential portfolio projects designed to demonstrate practical experience in Atlassian governance, DevOps, CI/CD, SRE, Cloud operations, metrics and cost optimization.

Each project is intentionally structured as if it could become an independent repository. Recruiters and technical reviewers can use these examples to validate how I organize problems, document decisions and translate operational experience into reusable engineering assets.

## Projects

| Project | Focus | What it demonstrates |
|---|---|---|
| [Atlassian Governance Audit Kit](./atlassian-governance-audit-kit) | Jira, Confluence, Bitbucket, plugins, governance | Platform administration, audit thinking, KPI extraction and reporting |
| [CI/CD Pipeline Standards Kit](./cicd-pipeline-standards-kit) | Jenkins, Shared Libraries, DevOps enablement | Pipeline governance, standardization, quality gates and reusable CI/CD design |
| [SRE Observability Metrics Kit](./sre-observability-metrics-kit) | SRE, MTTR, MTBF, SLA, dashboards | Reliability metrics, incident analysis and operational visibility |
| [Cloud Storage Optimization Kit](./cloud-storage-optimization-kit) | Docker storage, cost governance, hot/cold tiers | Storage optimization modeling inspired by the 4DS case study |

These examples are aligned with my professional background in DevTools, DevOps, Atlassian administration, CI/CD, SRE, MultiCloud operations, Power BI metrics, audit support and cost optimization.
